This project introduces a mobile (Android) port of [Lingoflow](https://github.com/ulasb/lingoflow).

Requirements:
- Users should be able to install the app as an apk.
- Users should be able to run the app locally, without a network connection, if desired. This means using an on-device model.
- Users should be able to run the app on a relatively fast and recent phone (At least the same specs or better as a Google Pixel 9)
- Users should be able to choose a cloud based model to use. This may require putting in authentication information.
- Otherwise, users should be able to get the functional and non-functional requirements and behavior as stated in the original Lingoflow project.
